## 6. Резинки — веса больше не выдуманные

Сценарий 6 (дом, только резинки+коврик, PPL 6 дней) — **все 24 сета** во всех 6 тренировках:

```
Push:   pushup@null, lateral_raise@null, knee_pushup@null, tricep_extension@null
Pull:   assisted_pullup@null, towel_row@null, table_row@null, curl@null
Legs:   bodyweight_squat@null, hip_thrust@null, lunge@null, plank@null
Push 2: lateral_raise@null, pushup@null, upright_row@null, tricep_extension@null
Pull 2: band_pulldown@null, towel_row@null, table_row@null, curl@null
Legs:   bodyweight_squat@null, hip_thrust@null, lunge@null, plank@null
```

Раньше `lateral_raise`/`curl`/`upright_row` получали случайное "2кг", хотя у пользователя физически нет гантелей. Теперь везде `null` — в UI это покажет "—" и даст пользователю самому вписать уровень сопротивления своей резинки, вместо вранья про конкретный вес.

Контрольная проверка (не должно сломаться для обычного зала): при полном оборудовании (штанга/гантели/скамья) вес по-прежнему передаётся как есть — юнит-тест `real gym equipment still keeps a literal weightKg` подтверждает это отдельно.
