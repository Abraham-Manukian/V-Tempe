# Screenshot Guide

Put README screenshots into this folder.

## Required files

- `onboarding.png`
- `home.png`
- `workout.png`
- `nutrition-menu.png`
- `nutrition-shopping.png`
- `sleep.png`
- `progress.png`
- `chat.png`
- `settings.png`

## Optional files

- `home-tablet.png`
- `workout-tablet.png`
- `chat-tablet.png`

## What each screenshot should show

- `onboarding.png`: onboarding UI with one profile step visible
- `home.png`: dashboard with summary cards and quick actions
- `workout.png`: workout details with checked sets and notes area
- `nutrition-menu.png`: meal plan tab with meals and macro summary
- `nutrition-shopping.png`: shopping list tab
- `sleep.png`: weekly sleep chart and tips section
- `progress.png`: analytics screen with charts
- `chat.png`: AI chat with several messages visible
- `settings.png`: settings/profile screen with AI mode section visible

## Recommended format

- Phone screenshots: around `1080x2400`
- Tablet screenshots: around `1600x2560`
- Prefer PNG
- Use full-screen captures
- Keep the same language across all screenshots
- Keep the same theme across all screenshots

## Updating the gallery

The main `README.md` already uses these file names.

If you want to improve the gallery later, replace the images in this folder while keeping the same names, then tell me to refresh the README captions or layout.
